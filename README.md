## PixelSnake
A simple game that tries to copy the Snake game basics

## Requiriments
You need [python](https://www.python.org/downloads/) and [pygame](https://www.pygame.org/wiki/GettingStarted#Further%20info%20on%20installation)

also if you want to compile the program you need [pyinstaller](https://github.com/pyinstaller/pyinstaller#installation) and type "pyinstaller game.spec"
